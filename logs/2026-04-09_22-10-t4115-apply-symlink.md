### Changes (summary)
- **`apply`**: Git-compatible symlink prechecks (`prepare_symlink_changes` / `path_is_beyond_symlink`), `index … mode` parsing (ignore second token when not octal mode), `can_apply_with_empty_preimage` only for `/dev/null` old paths, strip leading `a/`/`b/` from traditional headers after `-p0`, plan-then-execute worktree apply with in-memory pending state (no partial writes on failure), effective symlink mode for renames, delete errors match Git (`path: No such file or directory`), `--reject` stderr lines (`Rejected hunk`).
- **`diff-files`**: symlink-aware patch content (`read_link`), `index old..new mode` on modified hunks, stat fast path for symlinks when lstat matches index.
- **`clean`**: track paths by dentry name, not symlink target (fixes `clean -dfx` removing tracked `symlink -> .git`).
