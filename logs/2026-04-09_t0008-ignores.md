# t0008-ignores

## Failures (before)

- Test 393 `info/exclude trumps core.excludesfile`: `grit status --porcelain usually-ignored` printed `## master` before the path line; Git omits the branch header when pathspecs are given.
- Test 397 `symlinks not respected in-tree`: symlink `.gitignore` was followed; Git warns and ignores in-tree symlink ignore files.

## Fixes

- `grit/src/commands/status.rs`: only auto-enable porcelain branch line (`args.branch`) when there is no pathspec.
- `grit-lib/src/ignore.rs`: detect symlink `.gitignore` on disk, emit Git-style warning, return no rules (do not fall through to index blob).

## Validation

- `./scripts/run-tests.sh t0008-ignores.sh` → 398/398 pass.
- `cargo test -p grit-lib --lib` pass.
- `t12570-status-rename-copy.sh` re-run for porcelain regressions → pass.
