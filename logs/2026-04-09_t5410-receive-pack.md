# t5410-receive-pack

## Symptom

Test 5 (`receive-pack missing objects bypasses connectivity check`) failed: stderr had strict unpack error, `actual` contained `ng refs/heads/main missing necessary objects`.

## Root cause

With `--skip-connectivity-check`, receive-pack still called `unpack_objects` with `strict: true`. Thin packs reference parent commits not in the empty remote ODB; strict verification rejects "missing parent" and unpack failed.

## Fix

`grit/src/commands/receive_pack.rs`: set `UnpackOptions.strict` to `!args.skip_connectivity_check` so the skip-connectivity path matches Git (store objects without full closure check).

## Verification

- `cargo test -p grit-lib --lib` ŌĆö pass
- `./scripts/run-tests.sh t5410-receive-pack.sh` ŌĆö updates CSV to 5/5
