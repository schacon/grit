# t4069-remerge-diff (2026-04-09)

## Failures fixed

1. **Test 6 (`-S"not present"` under POSIX `/bin/sh`)**  
   The shell passes a single argv token `-Snot present`. Clap rejected it as an unknown short flag. Added `preprocess_show_argv` in `grit/src/main.rs` to split glued `-S<needle>` into `-S` + `<needle>` before parsing `git show`.

2. **Tests 9–11 (non-content conflicts)**  
   Merge records both `file/directory` and a synthetic `modify/delete` for the relocated `path~HEAD` blob. Upstream Git’s `show --remerge-diff` does **not** emit the extra modify/delete block (fixture in t4069). Filtered those descriptions in `write_remerge_diff` when `subject_path` matches a `file/directory` relocation path.
