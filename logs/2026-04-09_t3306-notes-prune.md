# t3306-notes-prune

## Failures addressed

1. **`git log` with notes** — Insert a blank line between commits when the previous commit had notes (matches upstream spacing without a trailing blank after the last commit).

2. **`git notes prune`** — Print pruned object full hex to stdout for `-n` / `-v` (not stderr), matching `notes.c`.

3. **`git reflog expire --expire=now HEAD`** — Expire the branch reflog when HEAD is symbolic; treat CLI `now` as `now+1` second so entries in the current second are pruned.

4. **`git prune`** — Stop marking every packed OID as reachable (was blocking removal of unreachable commits still in old packs).

5. **`pack-objects --all` (full repack)** — Enumerate ref closure only for non-incremental `--all`, not the entire ODB.

6. **`repack -d` after `repack -a`** — For plain full repack (`-a`, no cruft / `-A` / filter), delete every pack not in the new pack list so unreachable objects are not retained in superseded packs.
