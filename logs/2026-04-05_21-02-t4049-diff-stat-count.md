### Fixes implemented (current cycle)
- `grit diff --stat`/`--shortstat` counting:
  - Count `N files changed` by unique paths, not by rendered rows.
  - For binary entries, do not contribute insertion/deletion counts.
  - For unmerged entries, do not contribute insertion/deletion counts.
  - Only append insertion/deletion summary terms when at least one changed file exists.
- `grit diff --stat` rendering:
  - Added `Unmerged` marker rows to stat output path.
  - Count-column width now derives from rendered rows under `--stat-count`, reducing over-wide numeric padding.
- `grit reset --hard` working tree mode restore:
  - During checkout, non-executable blobs are now explicitly set to `0644` (not only executable entries set to `0755`).
  - This prevents executable mode leakage across test cases after hard reset.
