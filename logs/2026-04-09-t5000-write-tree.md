# t5000-write-tree

## Issue

All cases after the first failed with `cd: repo: No such file or directory` because test 1 ran `grit init repo && cd repo`, leaving the shell cwd inside `trash.../repo/`, while tests 2–5 still used `cd repo`.

## Fix

Drop the nested `repo` directory and use the harness-initialized repository at `TRASH_DIRECTORY` (same pattern as upstream `git/t/t0000-basic.sh`).

## Verification

`./scripts/run-tests.sh t5000-write-tree.sh` — 5/5 pass.
