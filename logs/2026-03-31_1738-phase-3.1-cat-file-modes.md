## Phase 3.1 - cat-file modes and tag dereference

- Implemented `gust cat-file -p` annotated-tag dereferencing so pretty-printing a tag shows the tagged object's content.
- Added safe recursive tag peeling with loop detection and explicit parsing of the tag `object <oid>` header.
- Preserved existing `-t`, `-s`, and `-e` mode behavior while keeping batch mode paths unchanged for this task.
- Added unit tests covering tag target parsing success and malformed tag headers.
