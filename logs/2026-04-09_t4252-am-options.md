# t4252-am-options

## Goal

Make `tests/t4252-am-options.sh` fully pass (Git `am` passthrough apply options and interrupted session resume).

## Changes

- `grit/src/commands/am.rs`: Added CLI flags matching `git am` passthrough to `git apply` (`--whitespace`, `--ignore-space-change`, `--ignore-whitespace`, `--directory`, `-p`, `-C`, `--exclude`, `--include`, plus `--reject` in `apply-opt`).
- Persist `rebase-apply/apply-opt` as shell-quoted argv (Git `sq_quote_argv` style); reload via `shell_words::split` on resume.
- `ApplySettings`: path resolution with `-p` and `--directory` (trailing path separator on directory prefix); optional include/exclude path filters.
- Hunk application: `-C` fuzz (trim leading/trailing context down to minimum), `--whitespace=fix` via `grit_lib::ws::ws_fix_copy_line`, ignore-whitespace modes for matching.
- `--reject`: apply successful hunks, write Git-style `diff a/… b/… (rejected hunks)` partial rejects, return failure without staging so index stays clean vs work tree (`git diff-files`).

## Validation

- `cargo test -p grit-lib --lib`
- `cargo fmt`, `cargo clippy -p grit-rs --fix --allow-dirty`
