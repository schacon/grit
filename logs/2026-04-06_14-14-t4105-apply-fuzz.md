### Implementation
- Added support for `git apply -C<n>`:
  - new `Args.context: Option<usize>` (`-C N`) in `grit/src/commands/apply.rs`.
  - context value threaded into hunk-matching mode.
- Improved hunk start resolution and offset tolerance:
  - nominal hunk starts are now clamped to file length (Git-compatible big-offset behavior).
  - hunk scanning considers minimum-context semantics when `-C` is active.
- Enabled fuzz fallback matching for `-C` flows:
  - when context is explicitly required, old-side matching can use ordered-subsequence fallback to tolerate altered context anchor lines while preserving in-order constraints.
