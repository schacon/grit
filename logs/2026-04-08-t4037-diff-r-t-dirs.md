# t4037-diff-r-t-dirs

## Symptom

Harness reported 0/2: setup failed because the second `git commit` never ran — only the root commit existed, so `HEAD^` was invalid in verify.

## Root cause

`git add -u` after replacing a tracked **file** with a **directory** (e.g. `ft` → `ft/` with `ft/5`) tried to `read()` the directory path and failed with `EISDIR` ("Is a directory").

## Fix

In `update_tracked` (`grit/src/commands/add.rs`): if the working tree path exists as a plain directory (not symlink) and is not an embedded repo (no `.git` child), remove the index entry — matching Git. Embedded repos still go through `stage_file` → gitlink.

## Ancillary

Removed a blank line between a `///` doc comment and `fn resolve_index_path` in `grit-lib/src/rev_parse.rs` so `cargo clippy -p grit-rs -- -D warnings` does not trip `empty_line_after_doc_comments` on that function.

## Verification

- `./scripts/run-tests.sh t4037-diff-r-t-dirs.sh` → 2/2 pass
- `cargo test -p grit-lib --lib` → pass
