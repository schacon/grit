### Implementation
- Added global CLI support for `--no-pager` / `--paginate` in `main` global-option parsing and applied it by setting `GIT_PAGER=cat`.
- Updated `diff` trailing-option recovery:
  - unknown trailing flags now emit `usage: git diff [<options>] [<commit>] [--] [<path>...]` and exit `129` (instead of being reinterpreted as revisions).
- Fixed `diff --check` exit-code semantics:
  - `--check` alone: exit `2` on check errors, `0` otherwise.
  - `--check --exit-code` (or `--quiet`): exit `3` when check errors exist; otherwise exit `1` for clean diffs and `0` for no diffs.
- Extended whitespace checker to detect leftover merge conflict markers in added lines:
  - `<<<<<<<`, `>>>>>>>`, `|||||||` with required trailing whitespace or EOL.
  - `=======` marker handling.
- Added `.gitattributes` support for `conflict-marker-size=<N>` via `grit-lib::crlf::FileAttrs` and used it in `diff --check` marker detection.
- Fixed `diff-tree --exit-code` behavior for pickaxe filtering:
  - `run_two_trees` now computes `has_diff` after applying `filter_entries_by_object`, so `-Snot-found` correctly returns success.
