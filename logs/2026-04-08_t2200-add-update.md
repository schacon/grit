# t2200-add-update

## Failures (before)

1. **error out when passing untracked path** — harness greps for `error: pathspec ...`; grit printed `fatal:`.
2. **add -n -u should not add but just report** — after replacing `foo` with a symlink to `top` and removing `top`, `Path::exists()` treated `foo` as absent; grit reported `remove 'foo'` in dry-run. Git keeps the symlink as present until explicitly removed.

## Fix

- `grit/src/commands/add.rs`: `update_tracked` pathspec error prefix `error:` (exit 128 unchanged).
- Same: use `fs::symlink_metadata(&abs_path).is_ok()` instead of `abs_path.exists()` when deciding update vs removal for `-u`.

## Validation

- `./scripts/run-tests.sh t2200-add-update.sh` → 19/19 pass.
- `cargo test -p grit-lib --lib` → pass.
