# grit show-index — implementation log

**Date:** 2026-04-01
**Branch:** impl/show-index

## Summary

Implemented `grit show-index`: reads a `.idx` pack index file from stdin and
prints one entry per line.

- Version-1 format: `<offset> <oid>`
- Version-2 format: `<offset> <oid> (<crc32>)`

## Changes

### `grit-lib/src/pack.rs`
- Added `ShowIndexEntry` struct (oid, offset, optional crc32).
- Added `show_index_entries(reader, hash_size)` — reads from any `Read` source,
  auto-detects v1 vs v2 by checking for the `0xff744f63` magic, delegates to
  `show_index_v1` or `show_index_v2`.

### `grit/src/commands/show_index.rs` (new)
- `Args`: `--object-format` flag (validates sha1 only, rejects anything else).
- `run()`: calls `show_index_entries` on stdin, prints each entry.

### `grit/src/commands/mod.rs`
- Added `pub mod show_index;`

### `grit/src/main.rs`
- Added `ShowIndex` variant and dispatch arm.
