  - implemented `--prune` and `--edge-aggressive` behavior using uninteresting-object propagation.
  - implemented `--indexed-objects` integration by reading stage-0 index entries and adding staged blobs and unchanged parent trees, with deferred scheduling to avoid duplicate emissions caused by preexisting tree walks.
