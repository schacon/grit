# t5609-clone-branch

## Goal

Make `tests/t5609-clone-branch.sh` pass (7/7).

## Failures (before)

1. `refs/remotes/origin/HEAD` missing after `clone -b` — `setup_remote_tracking_head` used `clone_ref_file_exists`, which only checked loose ref files, not `packed-refs`.
2. `clone -b bogus` and `clone -b branch` on empty repo succeeded — should fail like upstream Git (`Remote branch %s not found in upstream %s`).

## Fix

- `clone_ref_file_exists`: use `refs::read_raw_ref` → `RawRefLookup::Exists` so packed and reftable refs count.
- After `resolve_remote_name`, if `--branch` is set, require `refs/heads/<branch>` on the source or bail with Git-style message.

## Verification

- `./scripts/run-tests.sh t5609-clone-branch.sh` → 7/7
- `cargo check -p grit-rs`, `cargo test -p grit-lib --lib`
