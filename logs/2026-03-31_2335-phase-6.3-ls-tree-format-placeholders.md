## Phase 6.3 - ls-tree --format placeholders

- Implemented `ls-tree --format` placeholder expansion in `grit/src/commands/ls_tree.rs` for the v1 subset:
  - `%(objectmode)`
  - `%(objecttype)`
  - `%(objectname)`
  - `%(objectsize)` and `%(objectsize:padded)`
  - `%(path)`
- Added support for `%xNN` hex escapes and `%%` literal `%` in format strings.
- Added validation errors for malformed format strings (dangling `%`, invalid `%xNN`, unterminated `%(...)`, unknown fields).
- Ensured object type renders as `commit` for gitlink mode (`160000`) and computes blob object sizes for `objectsize` fields.
- Added CLI conflict handling so `--format` cannot be used with `--long` or `--name-only`/`--name-status`.
