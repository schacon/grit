# t1510-repo-setup

## Failure

Four harness cases failed: discovery aborted when `GIT_WORK_TREE` named a path that did not exist yet (ownership check `lstat` returned `ENOENT`), and absolute `GIT_WORK_TREE` with a missing final directory was rejected before open.

## Changes (`grit-lib/src/repo.rs`)

- **`ensure_valid_ownership`**: Treat missing work tree path as OK for the ownership pre-check (matches Git allowing not-yet-created work trees during setup).
- **`validate_git_work_tree_path`**: For absolute paths, require every prefix except the last normal path component to exist (leaf may be missing; t1510 #21).
