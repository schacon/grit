# 2026-04-01 19:50 — grit prune-packed

## Task

Implement `grit prune-packed` — removes loose objects that are already stored in a pack file.

## What was done

1. **Library** (`grit-lib/src/prune_packed.rs`): `prune_packed_objects(objects_dir, opts)` that:
   - Reads all local pack indexes to build a `HashSet<ObjectId>` of packed objects.
   - Walks two-char prefix loose object directories.
   - Deletes (or dry-run prints `rm -f <path>`) any loose object whose OID is in the packed set.
   - Tries to `rmdir` empty prefix directories after deleting.
   - Returns the list of paths removed.

2. **CLI** (`grit/src/commands/prune_packed.rs`): Parses `-n`/`--dry-run` and `-q`/`--quiet` flags, discovers the repository, and calls the library.
