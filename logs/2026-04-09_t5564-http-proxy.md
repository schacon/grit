# t5564-http-proxy

## Summary

- Extended `test-httpd` with `--proxy` / `--proxy-auth` (Apache-style proxy auth, CONNECT, absolute-form forwarding; fixed loopback hop auth and GET `Content-Length` for empty bodies).
- `tests/lib-httpd.sh`: wire proxy flags, export `LIB_HTTPD_PROXY` for hybrid `git` wrapper, atomic hybrid wrapper install, copy `socks4-proxy.pl`.
- Hybrid wrapper: use grit for HTTP(S) clone when `LIB_HTTPD_PROXY` is set; always use grit for `clone` with `-c http.proxy=...`.
- New `grit/src/http_client.rs`: `http.proxy` (HTTP forward with Basic proxy auth + `GIT_TRACE_CURL` redaction), SOCKS4 over Unix socket (percent-decode path once), `GIT_ASKPASS` for proxy password, early `http.proxy` validation for `clone -c`.
- `http_smart` / `clone` / `fetch`: plumb `HttpClientContext`; `main.rs` maps invalid proxy URL errors to `fatal:` exit 128.
- Workspace deps: `url`, `base64`.
