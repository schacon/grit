# t1006-cat-file

## Summary

Made `tests/t1006-cat-file.sh` fully pass (291/291).

## Changes

- **`pack-objects`**: `--unpack-unreachable` is optional; clap must not consume the trailing base-name (`git repack -A` / t1006 objects-filter setup).
- **`test-tool genrandom`**: Match Git’s LCG (`test-genrandom.c`) and `git_parse_ulong` size parsing including `k`/`m`/`g` suffixes (`10k` → 10240).
- **`test-tool path-utils file-size`**: Implement `stat` + print size (used by `test_file_size` in `test-lib-functions.sh`); was missing and broke `%(objectsize:disk)` expect construction.
- **`grit-lib` ODB**: Loose reads use path-implied OID without requiring payload hash match (swapped loose files); chmod loose objects to `0444` after write; `cat-file -e` uses `loose_object_plumbing_ok` for bogus-type loose objects.
- **`packed_delta_base_oid`**: Prefer newest pack first so `%(deltabase)` matches post-`repack` layout.
- **`cat-file`**: `collect_all_disk_object_entries` only scans primary `objects/` (no alternates duplicate lines for `%(objectsize:disk)`).
- **`fsck`**: Emit `error:` lines to stderr so `grep '^error: '` in t1006 matches.
