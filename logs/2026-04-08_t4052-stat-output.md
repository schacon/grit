# t4052-stat-output

## Summary

Implemented Git-compatible `--stat` layout in `grit-lib` (`diffstat` module): width budget from `COLUMNS` / `--stat-width`, `diff.statNameWidth` / `diff.statGraphWidth` merging, linear bar scaling, path truncation with `/` preference, optional ANSI-colored `+`/`-` runs, graph prefix width handling, and small bar slack for default `log --graph --stat`.

Wired through `diff`, `show`, `log` (including `--graph` path), `format-patch` (`--graph`, 72-column default, no implicit `diff.stat*` for patch emails), `merge --stat`, and fixed `diff` trailing-arg parsing order for `--stat-width` vs `--stat=`.
