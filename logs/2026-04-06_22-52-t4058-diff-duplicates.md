- `diff-tree`: added `--no-abbrev` parsing support and wired it to raw output abbreviation handling.
- `diff-tree` rename detection: self-path rename suppression for malformed duplicate trees now avoids synthetic `R100 path path` output.
- Duplicate-tree flattening:
  - `diff` tree flattener now tracks duplicate paths and returns corruption error when `GIT_TEST_CHECK_CACHE_TREE` is enabled.
  - `read-tree` and `reset` duplicate checks now return Git-style stderr (`error: corrupted cache-tree has entries not present in index`) instead of wrapped `corrupt object:` text.
- `checkout` branch switching now always refreshes index/worktree when switching branch names, even when both refs point to the same commit; this matches duplicate-tree fixture expectations for branch-state transitions.
- `main` top-level error renderer normalizes cache-tree corruption errors to Git-style prefixed stderr (`error: ...`) for all commands.
