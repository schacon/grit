# t7811-grep-open

## Changes (`grit/src/commands/grep.rs`)

1. **PATHSPEC_PREFER_CWD:** When there are no explicit pathspecs (non `--no-index`, non `--cached`, no tree-ish), add implicit `"."` so `resolve_pathspec` maps to the current prefix once (avoids `subdir/subdir` doubling).

2. **Shell pager argv:** Match Git `prepare_shell_cmd`: `sh -c 'pager "$@"' pager ...` when the pager string contains shell metacharacters and there are file arguments; plain `sh -c pager` when there are none.

3. **Pager cwd:** Run `--open-files-in-pager` with `current_dir` = the same as `git grep` (process cwd). T7811 `run from subdir` expects `pwd` to stay in the subdirectory and pager argv to list paths relative to it (e.g. `grep.c`).

4. **Paths passed to pager:** Use the same cwd-relative paths as normal `-l` output (`relative_path` via `pager_open_path: None` in `grep_content`); `--no-index` unchanged.

5. **Color:** For `-O`, pager file list uses `path_for_output` only (no ANSI), while normal `-l` output still uses `fmt_name` when color is on.
