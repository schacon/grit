## Implementation + validation

### Code changes
- `grit/src/commands/diff.rs`
  - Added implicit `--no-index` auto-detection for `git diff <path> <path>` when:
    - there is no repository, or
    - exactly two path operands are provided and at least one resolves outside the repo worktree.
  - Added helper functions:
    - `should_use_implicit_no_index(...)`
    - `is_path_outside_work_tree(...)`
    - `normalize_path_for_membership(...)`
  - Wired auto-detection early in `run()`:
    - if `Repository::discover(None)` fails and command shape is eligible, dispatches to `run_no_index(&args)`.
    - if repository exists but one of two explicit paths is outside worktree, dispatches to `run_no_index(&args)`.

- `grit/src/commands/diff_tree.rs`
  - In stdin handlers, `--quiet` now suppresses diff body while preserving stdin header echo behavior:
    - `process_stdin_commit(...)`: only prints diff when `!opts.quiet`.
    - `process_stdin_two_trees(...)`: only prints diff when `!opts.quiet`, returns `has_diff`.
  - This restores expected `diff-tree --quiet --stdin` semantics used by test 5.
