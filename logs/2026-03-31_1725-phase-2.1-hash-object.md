## Phase 2.1 - hash-object

- Implemented `grit hash-object` argument validation for:
  - `--stdin` with `--stdin-paths` (reject)
  - `--stdin-paths` with positional files (reject)
  - `--stdin-paths` with `--path` (reject)
  - `--no-filters` with `--path` (reject)
  - no input sources (reject)
- Added `--path` and `--no-filters` CLI flags for compatibility.
- Updated behavior so `--stdin` can be combined with positional files, processing stdin first, then files.
- Kept hashing/writing implementation in `grit-lib` object database paths (`Odb::hash_object_data`, `Odb::write`), with no `gix`/`git2`.
